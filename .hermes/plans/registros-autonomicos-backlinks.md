## 1. Andalucía

- **Organismo competente:** Agencia Andaluza de la Energía (Consejería de Industria, Energía y Minas)
- **Registro público de técnicos:** ✅ Sí — Registro Electrónico de Certificados de Eficiencia Energética
- **URL del registro:** https://www.juntadeandalucia.es/servicios/sede/tramites/procedimientos/detalle/1807.html
- **URL organismo:** https://www.agenciaandaluzadelaenergia.es/
- **Portal de trámites:** https://www.juntadeandalucia.es/servicios/sede/
- **¿Permite enlaces externos?** ❌ No se ha detectado una sección de "enlaces de interés" en el portal de certificación energética. La web usa Liferay y no muestra espacios para enlaces patrocinados o de colaboración.
- **¿Sección "enlaces de interés"?** ❌ No
- **¿Requisitos para ser técnico?** Titulación habilitante (arquitecto, arquitecto técnico, ingeniero, ingeniero técnico). Se requiere colegiación o habilitación profesional. No permite registro de plataformas como tal.
- **Oportunidad para CertificadoYa:** 🔴 **Baja** — El registro es individual para técnicos con titulación. No hay espacio visible para plataformas externas.

## 10. Comunidad Valenciana ✅ [ALTA PRIORIDAD]

- **Organismo competente:** IVACE (Instituto Valenciano de Competitividad Empresarial)
- **Registro público de técnicos:** ✅ Sí — "Registro de Técnicos Certificadores de Eficiencia Energética de Edificios"
- **URL del trámite:** https://www.gva.es/es/inicio/procedimientos?id_proc=20052
- **URL del portal de certificación:** https://gcee.aven.es/es/tecnicos-certificadores
- **URL de consulta pública:** https://www.gva.es/es/inicio/procedimientos?id_proc=G22776
- **¿Permite enlaces externos?** ✅ **Sí, expresamente.** El portal tiene:
  - **"Enlaces de interés"** — sección dedicada con enlaces a normativa, manuales y recursos.
  - **"Servicios para entidades colaboradoras de certificación"** — servicio específico para entidades externas.
  - **"Entidades colaboradoras de certificación"** — figura legal que permite a entidades externas colaborar.
- **¿Sección "enlaces de interés"?** ✅ **Sí**, claramente visible en la ficha del procedimiento (código SIA: 1997994).
- **¿Requisitos para ser técnico?** Titulación universitaria habilitante (arquitectos, ingenieros, ingenieros técnicos, graduados). Se requiere autorización al IVACE para comprobación de titulación o aportar título académico / certificado del colegio profesional. El registro de técnicos es **obligatorio** para poder registrar certificados.
- **Detalle importante:** La existencia de "Entidades colaboradoras de certificación" es única en España y podría permitir a CertificadoYa registrarse como entidad colaboradora para aparecer en el portal.
- **Oportunidad para CertificadoYa:** 🟢 **ALTA — Prioridad #1.** La C. Valenciana es la comunidad con mayor oportunidad porque:
  1. Tiene sección explícita de "Enlaces de interés" donde incluir certificadoya.es
  2. Tiene la figura de "Entidades colaboradoras de certificación"
  3. El portal GVA permite enlazar a plataformas externas
  4. IVACE gestiona directamente el registro

## Notas Metodológicas

- La verificación se realizó mediante peticiones HTTP directas con User-Agent de navegador estándar y parsing con BeautifulSoup4 + lxml.
- Muchos portales autonómicos usan Liferay Portal con renderizado JavaScript del lado del cliente, lo que limita la extracción de contenido mediante scraping tradicional.
- Las URLs marcadas como "error 404" o "no accesible" pueden estar activas pero requerir JavaScript para su renderizado completo.
- La información sobre requisitos de colegiación se basa en el RD 390/2021 de ámbito nacional y en la documentación específica de cada CCAA cuando estaba disponible.
- **Próximo paso recomendado:** Contactar directamente con cada CCAA identificada como oportunidad para confirmar la política de enlaces externos y entidades colaboradoras.
