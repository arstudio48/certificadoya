¿Es obligatorio el certificado energético para vender?

Sí, siempre es obligatorio. El Real Decreto 235/2013 establece que es obligatorio disponer de certificado energético para cualquier transmisión patrimonial de un inmueble.

Esto incluye:

  • Venta de vivienda (nueva o usada)
  • Donación de inmueble
  • Herencia de inmueble
  • Permuta de inmueble
⚠️ IMPORTANTE: El notario NO puede formalizar la escritura de compraventa sin el certificado energético. Es un documento obligatorio como la nota simple del registro.

¿Qué pasa si vendo sin certificado energético?

🚫 SIN CERTIFICADO:
  • El notario rechaza la escritura
  • La venta no se puede formalizar
  • Puedes recibir una multa de 300€ a 6.000€
  • El comprador puede demandarte por daños

¿Cómo obtener el certificado para vender?

Paso 1: Solicita presupuesto

En CertificadoYa obtienes un presupuesto instantáneo. Solo necesitas la dirección y tipo de vivienda.

Paso 2: Visita del técnico

Un técnico certificado visitará la vivienda para las mediciones. Dura 30-60 minutos.

Paso 3: Emisión del certificado

El técnico calcula la eficiencia energética con software oficial y emite el certificado.

Paso 4: Registro

El certificado se registra en el órgano competente. Una vez registrado, puedes formalizar la venta.

Paso 5: Entrega al comprador

El certificado energético se entrega al comprador junto con la escritura de compraventa.

¿Qué letra tendrán mis pisos?

La letra depende de las características del inmueble:

  • A (máxima eficiencia): Nuevos con aislamiento, placas solares, etc.
  • B: Nuevos con buena eficiencia
  • C: Renovaciones recientes
  • D: Pisos con 15-20 años sin renovar
  • E: Pisos antiguos con mejoras parciales
  • F: Pisos antiguos sin mejoras
  • G (mínima eficiencia): Pisos muy antiguos sin aislamiento

Preguntas frecuentes sobre certificado energético para venta

¿Es obligatorio el certificado energético para vender un piso?

Sí. Es obligatorio para cualquier transmisión patrimonial (venta, donación, herencia, permuta). Sin él, el notario no puede formalizar la escritura.

¿Qué pasa si vendo sin certificado energético?

El notario rechaza la escritura, la venta no se puede formalizar y puedes recibir una multa de 300€ a 6.000€.

¿Cuánto cuesta el certificado para vender?

Desde 45€ para pisos pequeños hasta 200€ para chalets. El precio incluye todo: visita, mediciones, emisión y registro.

¿Quién paga el certificado energético en una venta?

Siempre lo paga el vendedor. Es el propietario del inmueble y quien necesita el certificado para formalizar la venta.
