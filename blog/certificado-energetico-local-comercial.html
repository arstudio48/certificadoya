¿Necesito certificado energético para un local comercial?

Sí. El Real Decreto 235/2013 establece que es obligatorio disponer de certificado energético para cualquier inmueble que se venda o alquile, incluyendo:

  • Locales comerciales (tiendas, restaurantes, oficinas en local)
  • Naves industriales
  • Garajes (cuando se venden/alquilan individualmente)
  • Trasteros (cuando se venden/alquilan individualmente)
  • Terrenos (cuando tienen edificaciones)

El precio es mayor que para viviendas porque los locales suelen tener:

  • Mayor superficie → más mediciones
  • Instalaciones complejas → climatización industrial, cámaras frigoríficas
  • Uso mixto → pueden tener zona de almacenaje, zona de venta, zona de oficinas

Particularidades del certificado para locales

1. Consumo energético diferente

Un local comercial tiene un perfil de consumo muy diferente a una vivienda:

  • Horario de uso: 8-12 horas diarias (vs 24h en vivienda)
  • Iluminación: más intensa y durante más horas
  • Climatización: puede ser más potente (gente, puertas abiertas)
  • Equipamiento: electrodomésticos comerciales, maquinaria

2. Software de cálculo

El técnico puede usar diferentes software según el tipo de local:

  • CE3X: Para locales pequeños y simples
  • HULC: Para locales complejos o con uso mixto
  • Certificate: Para locales de nueva construcción

3. Registro

El certificado de locales se registra en el mismo órgano que el de viviendas. No hay diferencia en el proceso de registro.

¿Qué documentos necesito?

  • Referencia catastral del local
  • Plano de distribución (si no lo tienes, el técnico lo mide)
  • Certificado de compatibilidad urbanística (recomendable)
  • Escritura de propiedad (opcional pero recomendable)

¿Cómo mejorar la letra de un local comercial?

Los locales suelen tener letra E, F o G. Las mejoras más efectivas son:

  • LED en toda la instalación: reduce consumo un 60%
  • Climatización eficiente: bomba de calor o sistema centralizado
  • Aislamiento de fachada: especialmente si tiene cristalera
  • Sistema de gestión energética: termostatos programables

Preguntas frecuentes sobre certificado energético para locales

¿Un local comercial necesita certificado energético?

Sí. Cualquier inmueble que se venda o alquile necesita certificado energético, incluyendo locales comerciales, oficinas y naves industriales.

¿Cuánto cuesta el certificado para un local?

Desde 80€ para locales pequeños hasta 500€ para naves industriales. El precio depende de la superficie y complejidad del inmueble.

¿El certificado de local es igual que el de vivienda?

Sí, es el mismo tipo de documento. La diferencia está en el cálculo del consumo energético, que tiene en cuenta el uso comercial del inmueble.

¿Puedo alquilar un local sin certificado energético?

No. Alquilar sin certificado energético puede acarrear sanciones de 300€ a 6.000€. Es obligatorio entregarlo al inquilino.
