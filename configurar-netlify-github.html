🚀 Conectar GitHub + Netlify

Sigue estos pasos para que los deploys sean automáticos:

  1. Ve a app.netlify.com
  2. Inicia sesión con aparejado@me.com
  3. Ve a certificadoya-es → Site configuration → Build & deploy
  4. En "Build settings", haz clic en "Link to Git repo"
  5. Conecta con GitHub y busca arstudio48/certificadoya
  6. En "Build command": déjalo vacío (es HTML estático)
  7. En "Publish directory": . (el raíz del repo)
  8. En "Functions directory": netlify/functions
  9. Añade variable de entorno: SUPABASE_SERVICE_KEY = sb_secret_7LGmViutdXYU16hwvAqlJg_hZjhfcLe
  10. Haz clic en "Deploy site"

Una vez conectado, cada vez que hagas git push a main, Netlify despliega automáticamente.
